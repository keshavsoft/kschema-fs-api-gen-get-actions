About KSchema

What KSchema does, its key benefits, and supported scaffolding commands.

Overview

`kschema-fs-api-gen-get-actions` is a lightweight, high-performance CLI scaffolding package built for Node.js. It helps developers generate standard, clean, and reusable Express.js controller mappings, database access layers (DALs), route definitions, and validation files based on their database schemas.

Instead of manually copy-pasting code blocks and risking error prone manual imports, KSchema handles everything dynamically using metadata, saving hours of development time.

Core Benefits

⚡ Fast Scaffolding

Scaffolds full-stack endpoint routes, validators, data access layers (DALs), and controller maps in milliseconds.

🧩 Dynamic Architecture

Configured using actions.json. Drop in a new command mapping, and it is instantly loaded by the CLI.

🔒 Duplicate Protection

Includes built-in validation checks protecting files from unintended duplication and duplicate imports.

Supported Commands

Default generated Express.js endpoints boilerplate supported by the generator:

Command Name HTTP Method Description
ShowAll GET Generates full controller, dal, route, and validator for fetching all rows in a database table.
Insert POST Generates table row entry creation handler with data type & validation rules.
Distinct GET Generates table handler fetching distinct filter values.
