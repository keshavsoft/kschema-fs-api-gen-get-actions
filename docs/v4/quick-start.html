Quick Start

How to install, execute, and extend scaffolding commands inside your project.

Installation

You can run KSchema globally or run it directly using `npx` without local installation.

bash
npm install -g @keshavsoft/kschema-api-gen-actions # Run the dynamic action generator command npx @keshavsoft/kschema-api-gen-actions ShowAll

Generated Output Structure

When running commands, KSchema creates clean, isolated directories matching the target action, preserving your existing core folders:

directory
project/ ├── end-points.js │ ├── ShowAll/ │ ├── controller.js │ ├── dal.js │ ├── route.js │ └── validation.js

Adding New Actions

You can easily add new actions in three simple steps:

steps
1. Create action logic file under tasks/actions/ e.g., tasks/actions/update.js 2. Add a new configuration entry in actions.json: { "cmd": "update", "file": "update", "exportFile": "Update", "group": "PostMethods", "description": "Generate Update POST action" } 3. Done! The command is immediately available in the CLI.
