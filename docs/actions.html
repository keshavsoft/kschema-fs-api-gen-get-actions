Required Workflow Secrets

To authorize cross-repository API requests and deploy builds, configure these secrets in repository settings:

Secret Key Target Role
NPM_TOKEN Authorization token for publishing packages to npmjs.org.
REPO_DISPATCH_TOKEN GitHub Personal Access Token (PAT) with repository scopes to push code and trigger downstream dispatches.
